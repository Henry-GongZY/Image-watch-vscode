## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Compile TypeScript:
   ```bash
   npm run compile
   ```

3. Press F5 to run the extension in development mode

## Commands

- `image-watch-vscode.helloWorld`: Show hello world message
